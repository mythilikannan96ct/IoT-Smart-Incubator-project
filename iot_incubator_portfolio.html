System Design

Full-Stack IoT Architecture

A layered hardware-software system built for reliability — with PID temperature control, MQTT telemetry, OLED status display, and a live web dashboard. Runs 24/7 on a Raspberry Pi 4.

🌡️

Sensor Layer

DHT22 for air temp + humidity, DS18B20 for egg surface probe. Rolling 5-sample average for noise rejection. Retry logic with hardware fault detection.

DHT22 · DS18B20 · 1-Wire · GPIO
⚙️

Control Layer

PID temperature controller with anti-windup. Bang-bang humidity control. Egg turning scheduler with lockdown detection. All relay states managed via RPi GPIO.

PID · 4-CH Relay · L298N · Threading
📡

Network Layer

MQTT publishes sensor + relay telemetry every 10 seconds to local broker. Flask REST API for remote control. Live JSONL data log with CSV export.

MQTT · Flask · Paho · REST API
🖥️

Display Layer

SSD1306 128×64 OLED shows live readings, day counter, and alarm state. Web dashboard with Chart.js trend graphs. Buzzer for alarm notifications.

SSD1306 · I2C · Chart.js · PIL
🔄

Turner Module

Standalone Arduino Nano with L298N motor driver, limit switches, RTC DS3231, and LCD. Communicates with Pi via UART JSON. Independent fallback operation.

Arduino · L298N · DS3231 · ISR

class PIDController:
    """
    PID controller for heater duty-cycle regulation.
    Output: 0–100% duty cycle.
    """

    def __init__(self, kp=2.0, ki=0.5, kd=1.0,
                 setpoint=37.5):
        self.kp        = kp
        self.ki        = ki
        self.kd        = kd
        self.setpoint  = setpoint
        self._integral = 0.0
        self._prev_err  = 0.0
        self._prev_time = None

    def update(self, measured: float) -> float:
        now = time.time()
        dt  = (now - self._prev_time) if self._prev_time else 1.0
        self._prev_time = now

        error = self.setpoint - measured

        # Anti-windup integral clamp
        self._integral += error * dt
        self._integral  = max(-50, min(50, self._integral))

        derivative = (error - self._prev_err) / dt
        self._prev_err  = error

        output = (self.kp * error
                + self.ki * self._integral
                + self.kd * derivative)

        return max(0.0, min(100.0, output))
class SensorManager:
    """Rolling-average sensor reads with retry & validation."""

    def get_reading(self) -> Optional[SensorReading]:
        temp, humid = self.read_dht22()
        if temp is None:
            log.error("Sensor read failed")
            return None

        egg_temp = self.read_ds18b20()   # 1-Wire probe
        reading  = SensorReading(temp, humid, egg_temp)

        self._readings.append(reading)
        if len(self._readings) > self._window:
            self._readings.pop(0)
        return reading

    def get_smoothed(self) -> Optional[SensorReading]:
        # 5-sample rolling average
        n     = len(self._readings)
        avg_t = sum(r.temperature_c for r in self._readings) / n
        avg_h = sum(r.humidity_pct  for r in self._readings) / n
        return SensorReading(round(avg_t, 2), round(avg_h, 1), None)

    def read_dht22(self):
        for attempt in range(3):
            h, t = Adafruit_DHT.read_retry(SENSOR_DHT22, PIN_DHT22)
            if t and 10 <= t <= 50:
                return round(t, 2), round(h, 1)
        return None, None
class MQTTPublisher:
    """Publishes telemetry to MQTT broker (Mosquitto)."""

    def publish_telemetry(self, reading, relays, alarms, config):
        # incubator/sensors
        self.publish("sensors", reading.to_dict())

        # incubator/relays
        self.publish("relays", relays.to_dict())

        # incubator/status
        self.publish("status", {
            "day":           config.day_number,
            "days_remaining": config.days_remaining,
            "is_lockdown":   config.is_lockdown,
            "egg_type":      config.egg_type.value,
            "alarms":        asdict(alarms),
            "timestamp":     datetime.now().isoformat(),
        })

// Arduino egg turner — 4h auto-turn with limit switches

void performTurn() {
  Serial.println("TURN_START");

  // Alternate direction each turn
  bool goRight = (state.totalTurns % 2 == 0);

  if (goRight && !state.atRightEnd)  motorForward();
  else if (!goRight && !state.atLeftEnd) motorReverse();
  else { goRight ? motorReverse() : motorForward(); }

  // Wait for limit switch or 30s timeout
  unsigned long startMs = millis();
  while (state.direction != STOP
      && !limitL_triggered && !limitR_triggered
      && (millis() - startMs) < TURN_TIMEOUT_MS) {
    delay(50);
  }

  softStop();         // ramp down PWM
  state.totalTurns++;
  beep(1, 80);
}

// ISR — fires instantly when limit switch hits
void ISR_limitL() { limitL_triggered = true; }
void ISR_limitR() { limitR_triggered = true; }

Capabilities

What It Does

🌡️

PID Temperature Control

PID algorithm with anti-windup. ±0.5°C tolerance band. Heater and fan coordinated for fast, stable regulation.

💧

Humidity Management

On/off ultrasonic humidifier control. Auto-increases RH during lockdown phase per egg species profile.

🔄

Egg Turning Schedule

Motorised turning every 4 hours. Alternating direction. Auto-stops in lockdown. Arduino backup module for reliability.

📡

MQTT Telemetry

Publishes to 3 topics every 10 seconds. Integrates with Node-RED, Home Assistant, Grafana, or any MQTT dashboard.

🌐

Web Dashboard

Flask REST API + live Chart.js dashboard. Adjust setpoints, start/stop, view 24h trends remotely via browser.

🚨

Alarm System

Buzzer + MQTT alert on temp/humidity out-of-range. Sensor failure detection with safe fallback state.

🥚

5 Egg Species Profiles

Chicken, duck, quail, goose, turkey — each with correct incubation days, temp, humidity, and lockdown day pre-loaded.

💾

Data Logging

JSONL rolling log for every sensor cycle. REST endpoint to query last N hours. Persistent across reboots.

Setup

Get Running in 5 Minutes

# Clone the repo
$ git clone https://github.com/YOU/iot-incubator.git
$ cd iot-incubator

# Enable hardware interfaces
$ sudo raspi-config
→ Interface Options → I2C → Enable
→ Interface Options → 1-Wire → Enable

# Install Python dependencies
$ pip install -r requirements.txt

# Install & start MQTT broker
$ sudo apt install mosquitto mosquitto-clients -y
$ sudo systemctl enable --now mosquitto

# Run the controller
$ python incubator_controller.py

# Or run with web dashboard
$ python web_dashboard.py
→ Open http://<pi-ip>:5000 in your browser

# Arduino turner (optional)
→ Open egg_turner/egg_turner.ino in Arduino IDE
→ Upload to Arduino Nano → connect to Pi via USB serial
